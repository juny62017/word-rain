# Word Rain

Word Rain is a small WEB PLAYABLEg game built with plain HTML, CSS, and JavaScript.

Words fall from the top at random positions.
Type the first letter to lock onto a word, then finish typing the rest of it before it reaches the bottom.
Missing a word costs one life. 
The game ends when all lives are gone.


## THEME 

IT IS A ENDLESS GAME
AS THERE IS NO END HERE 

This project is part of the Endless concept
as words keep spawning forever.... and there is no ultimate level....or winning point.
The game randomly chooses words from the list and an algorithm slowly raises the speed at which they fall
and decreases the spawning delay according on the player’s score and survival time.

## Features

- random words falling
- Once you type the first letter of word.. it lockes it
- Lives and game over 
- Score tracking according to your game play
- Endless difficulty 
- Faster fall speed over time
- Faster word spawning as the score increases
- Special multiplier words
- Temporary 2x score bonus
- Start screen and restart flow


## Controls

- Type letters to target and destroy words
- Press Esc to release the current word
- Press Enter to start or restart
- You can also click the canvas to start or restart

## How to Run

  Go to demo link
  START PLAYING 



The game should work in modern desktop browsers such as Chrome, Edge, Firefox
A physical keyboard is recommended.
